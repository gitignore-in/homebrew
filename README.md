# homebrew-gitignore-in

Homebrew Installation of [gitignore.in](https://github.com/gitignore-in/gitignore-in).

## Installation

```console
$ brew tap gitignore-in/gitignore-in
$ brew install gitignore-in
```

## Update

```console
$ ./generate.sh
```
